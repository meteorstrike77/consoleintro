Console Project

A console is a component of a website that is accessed by right clicking, clicking Inspect, and then clicking Console. A console is typically used to test code on a smaller screen instead of testing it on a project itself. To print something on a console, use the console.log function.

If you open the console of this webpage, you will see many strings of text. Check it out!

What is Visual Studio?

Visual Studio is an offline IDE (Integrated Development Environment) created by Microsoft. To set it up, go to the Visual Studio website, click the link, and then extract the files. Visual Studio will then be downloaded. You can use many different coding languages in Visual Studio, such as HTML, CSS, and JS. It can be used to create amazing webpages and other types of coding.
